## Списки

Чтобы добавить не нумсерованные пункты, необходимо пункты выделить звёздочкой (*), например вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать, например вот так:
1. Первый пункт
2. Второй пункт
3. Третий пункт
